# PDF 制作规格书：「你的人生故事卡包」

> 版本：V1.0 | 日期：2026-06-27 | 可直接交付排版师/设计师执行
> 目标格式：高清 PDF（线上售卖 + 用户自行打印）

---

## 一、整体规格

| 项目 | 规格 | 备注 |
|------|------|------|
| **成品尺寸** | A5（148 × 210 mm） | 比手机大一点，适合捧读，也方便打印在 A4 纸（一张 A4 可打两张 A5） |
| **页数** | 约 110-120 页 | 含封面、扉页、引言、10 个主题分隔页、100 题正文、收尾页 |
| **出血线** | 3 mm | 方便印刷厂裁切，自打印用户可忽略 |
| **文件格式** | PDF（CMYK 色彩模式，300 dpi） | 印刷版用 CMYK，线上售卖版可同时提供 RGB 版本 |
| **装订建议** | 线圈装/活页装 | 用户可平摊书写，也可抽出单张送人 |

---

## 二、色彩与字体

### 色彩方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景主色 | #FAF8F5（暖米白） | 模拟旧纸张的温暖感 |
| 正文文字 | #3D322C（深棕灰） | 不用纯黑，降低攻击性 |
| 问题文字 | #2C241E（深棕） | 比正文稍深，突出视觉层级 |
| 灵感提示 | #9C8B7E（灰驼） | 弱化，像耳边轻语 |
| 主题标签色 | #C4A882（暖驼） | 分隔页的主题标识 |
| 点缀色 | #D4A574（琥珀） | 极小面积使用：页码、小符号 |
| 留白区边框 | #E8E0D5（浅驼） | 极淡的分隔线 |

### 字体方案

| 用途 | 字体 | 字号 | 行距 |
|------|------|------|------|
| 主题标题 | 思源宋体 Bold | 24 pt | 1.5 倍 |
| 主题释义 | 思源宋体 Regular | 11 pt | 1.6 倍 |
| 问题正文 | 苹方/思源黑体 Medium | 14 pt | 1.8 倍 |
| 灵感提示 | 苹方/思源黑体 Light | 10 pt | 1.6 倍 |
| 页码 | 苹方 Regular | 9 pt | — |
| 情绪引导文字 | 思源宋体 Italic | 11 pt | 1.8 倍 |

## 三、页面结构（完整顺序）

```
[封面]
    ↓
[扉页 - 空白]（模拟翻书的仪式感）
    ↓
[引言页]「这不是一场考试…」
    ↓
[使用说明]年龄段建议 + 使用场景
    ↓
[主题 1 分隔页] 觉醒时刻
    ├─ 题 1 + 灵感提示 + 书写区
    ├─ 题 2 + 灵感提示 + 书写区
    ├─ ...
    ├─ 题 10 + 灵感提示 + 书写区
    └─ [中间庆祝页]「你已经完成 1/10 啦」
    ↓
[主题 2 分隔页] 至暗时刻
    └─ ... 同上
    ↓
... 重复 10 个主题 ...
    ↓
[收尾页 - 庆祝]「恭喜你，走完了这段旅程」
    ↓
[送出指引页]「如果你想送给一个人…」
    ↓
[收件人阅读指南页]「如果你收到这本故事…」
    ↓
[封底]
```

## 四、单题页面布局规范

以 A5（148 × 210 mm）为画布：

```
┌─────────────────────┐
│  边距 15mm          │  ← 安全区域
│                     │
│  # 1               │  ← 题号，放在左上角，小字
│                     │
│  有没有一个瞬间，     │
│  你突然意识到          │  ← 问题正文，14pt，行距 1.8
│  「世界不是围着        │     居中偏左，占页面 1/3
│   我转的」？          │
│  那一刻的心境          │
│  是什么样的？          │
│                     │
│  ————— 灵感提示 —————  │
│  不一定是重大挫折，     │  ← 灵感提示，10pt，灰驼色
│  可能是一次独自碰壁、   │     与问题之间留白 8mm
│  一次看懂大人的不易…   │
│                     │
│ ┌─────────────────┐ │
│ │                 │ │
│ │   书写区         │ │  ← 留白区域，占页面 1/2
│ │   （淡色横线     │ │     浅驼色分隔线标记
│ │    或不划线）    │ │
│ │                 │ │
│ │                 │ │
│ └─────────────────┘ │
│                     │
│         • 01        │  ← 页码，右下角
└─────────────────────┘
        边距 15mm
```

### 布局规则

| 元素 | 位置 | 约束 |
|------|------|------|
| 题号 | 左上角，距页边 15mm | 10pt，琥珀色 |
| 问题正文 | 顶部 1/3 区域，左对齐 | 14pt，行距 1.8 倍，最多 6 行 |
| 灵感提示分隔线 | 问题与提示之间 | 细横线，长度 30mm，浅驼色 |
| 灵感提示 | 问题下方 8mm | 10pt，灰驼色，缩进 5mm，最多 4 行 |
| 书写区 | 页面底部 1/2 | 淡横线（0.3pt）或不划线，留白为主 |
| "可跳过"提示 | 书写区右下角 | 8pt，「如果暂时不想写，跳过它→」，浅驼色 |
| 页码 | 右下角，距页边 15mm | 9pt |

## 五、特殊页面设计

### 主题分隔页

每 10 题结束后插入一个分隔页：

```
┌─────────────────────┐
│                     │
│                     │
│      觉醒时刻        │  ← 主题名，思源宋体 Bold，28pt
│                     │
│   看清世界、          │  ← 主题释义，11pt，居中
│   重塑自我的瞬间      │
│                     │
│   ──── ● ────       │  ← 装饰元素，琥珀色
│                     │
│    第 1-10 题        │
│                     │
│                     │
└─────────────────────┘
```

### 中间庆祝页（每 10 题后）

```
┌─────────────────────┐
│                     │
│   ✦                │
│                     │
│   你已经完成          │
│   1/10 的人生故事    │
│                     │
│   不需要一次写完      │  ← 情绪引导文字
│   歇一歇              │
│   喝杯热茶            │
│   随时回来            │
│                     │
│                     │
└─────────────────────┘
```

### 收尾庆祝页（100 题全部完成后）

```
┌─────────────────────┐
│                     │
│      恭喜你          │
│                     │
│   这不是一场考试      │
│   没有分数            │
│   但你已经做了一件     │
│   很勇敢的事：        │
│                     │
│   你选择              │
│   被看见              │
│                     │
│                     │
└─────────────────────┘
```

### 送出指引页

> 告诉用户如何把写完的卡包送给别人。

核心文案方向：
- 「你可以把整本送出去，也可以只撕下某一页」
- 「如果你不确定对方会不会认真看——先送一页试试」
- 「收到的人可能不知道怎么回应，建议附上一张小纸条：『不用回送什么，我只是想让你知道我的这一面』」

### 收件人阅读指南页（重要！被忽略的设计）

这是两份设计文档都遗漏的关键页面：

> 当收件人打开这本被写满的故事卡包时，他/她需要指引：
> - 「这是一份礼物，不是一个任务——你不需要回答任何问题」
> - 「如果你读到某页哭了，那是正常的」
> - 「如果你读完不知道该说什么，『谢谢』就够了」
> - 「这本故事的主人选择把最真实的一面给你，请小心对待」

## 六、制作工具与流程

| 阶段 | 工具 | 目的 |
|------|------|------|
| **黑白文字版** | Markdown → Typora/Word 导出 PDF | 先验证问题质量，不投资排版 |
| **设计版** | Canva / Figma / InDesign | 正式排版，含视觉设计 |
| **印刷测试** | 打印 1 本实体样 | 检查字号、留白、装订效果 |
| **最终输出** | 导出 CMYK PDF（300dpi） | 印刷版 + 线上版 |

## 八、Non-goals（这个规格书不涉及）

- 不做小程序/网页端的 UI 设计
- 不做实体卡片的材质和印刷方案（那是 PDF 验证通过后的事）
- 不做品牌 VI 体系（先验证内容再建品牌）
- 英文版/多语言不做
