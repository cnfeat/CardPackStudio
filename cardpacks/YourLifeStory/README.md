## 这是什么

一本 A6 尺寸的问答卡片集。不是问卷，不是考试，不是自传——是 100 个精心设计的问题，引导你回望自己的人生，把零散的瞬间变成可以被看见、被理解、被记住的鲜活故事。

---

## 如何使用

### 基础玩法：慢慢写
- 没有时间限制，没有字数要求
- 写一句话也行，写三页也行
- 今天写三道，三个月后再翻开也行
- 跳过不是放弃，是尊重自己的节奏

### 进阶玩法：传记深聊
挑几张卡片 + 手机开录音 + 跟在乎的人深聊，结束后把录音转成文字稍作整理，就是一份专属的**个人传记草稿**。送给对方那一刻，价值感拉满。
