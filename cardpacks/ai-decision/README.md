## 📦 卡包内容

### 四大创业阶段

| 阶段 | 卡片数量 | 核心主题 | 关键转变 |
|---|---|---|---|
| **想法** | 25张 | 找到真正值得解决的问题 | 从"能做"到"该做" |

### 双层卡片结构

```
🎴 001

[问题层：独立完整的核心问题，约35字]

[扩展层：3个延伸小问连续不分行，约100字]
```

#### 问题层
- **定位**：独立完整的核心问题
- **字数**：约35字
- **作用**：引发思考的钩子，直击决策本质

#### 扩展层  
- **构成**：3个延伸小问，连续出现不分行
- **字数**：约100字
- **角度**：个人经验、理论原则、批判反思三个维度

## 🚀 使用方法

### 1. 下载PDF（免费）
直接下载PDF版本进行阅读和学习。

### 2. 导入插件（推荐）
将 `questions.json` 文件导入「日课一问」插件平台：
1. 下载「日课一问」Chrome插件
2. 进入插件设置
3. 选择"导入卡包"
4. 选择 `questions.json` 文件
5. 开始每日思考

### 3. 最佳实践
- **定时思考**：每天固定时间进行卡片思考
- **记录反思**：将思考过程和决策结果记录下来
- **团队共享**：与创业团队共享，统一决策框架
- **定期回顾**：每月回顾思考历程，调整决策方法

## 🎨 设计理念

### 卡片法方法论
- **简洁性**：每张卡片独立完整，信息密度适中
- **可操作性**：直接指导行动，不空谈理论
- **迭代性**：可通过用户反馈持续优化
- **扩展性**：可延伸到其他决策场景

### AI时代适配
- **技术中立**：不依赖特定技术栈，关注决策本质
- **快速迭代**：适应快速变化的商业环境
- **人机协作**：强调人类判断在AI时代的核心价值
